# AgentViz

## Vision

AgentViz is an open-source observability platform for AI agents.

It helps developers understand, debug, and evaluate AI agent executions through workflow visualization, tool inspection, memory tracking, and execution timelines.

The goal is to become the Chrome DevTools for AI Agents.

---

## Problem Statement

Developers struggle to understand:

- Why an agent chose a tool
- What payload was sent
- What response was returned
- What memory changed
- Why execution failed

AgentViz makes agent behavior transparent.

---

## Target Users
